Introduction

HTML (HyperText Markup Language) is a markup language that is fundamental in the construction of a website. Although there are other pieces of code that can go into making a website(i.e. CSS and Javascript), HTML is the only code that is vital to all websites. Without HTML, there would be no website to work with.

As the name implies, HTML is a markup language. This is because HTML involves content that is marked up with certain code that indicate how each bit of content within the website should be presented. The code that marks up a webpage's content in this manner is refered to as an element.

Listed below are some examples of HTML elements and how they would be seen in an HTML file:

  • HTML Element(<html></html>)
  • Head Element(<head></head>)
  • Body Element(<body></body>)
  • Paragraph Element(<p></p>)
  • Division Element(<div></div>)

As seen above, HTML elements commonly consist of two tags. The first tag is known as an opening tag while the last tag is known as an ending tag. Something like <p> would be an opening tag while </p> would be a closing tag. Notice that the way to distinguish between the two is the dash that comes right after the starting angle bracket. Note that some elements only consist of one tag(i.e <br />). These tags are known as self closing tags.

Basic Structure

Listed below is the basic structure that an HTML page generally follows:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
<link rel="stylesheet" href="style.css" />
</head >
<body>
</body>
</html>

Here is a brief explanation of what each of these elements mean:

  • The <!DOCTYPE html> declaration indicates which verison of HTML the code is written in. In this case, HTML5 is being used.
  • The <html> element is referred to as the root of the HTML document and it will contain all other HTML elements except the <!DOCTYPE html> declaration.
  • The <head> element contains the meta data of the webpage which is data that is usually not visible on the website itself.
  • The <title> element contains the title of the webpage. This title is displayed on the website tab at the top of the web browser.
  • The <link> element allows us to connect a CSS file that contains styling for the page. CSS will not be covered in this documentation page. Nonetheless, it is a good idea to be aware of the link element and what it looks like.
  • The <body> element contains all the visible elements of the webpage.
Heading Elements

Heading elements contain the headings that are within a webpage. There are six types of heading elements and they range from h1 to h6.

The h1 element contains a heading that should describe the overall purpose of the page. There should only be one h1 within a webpage.

The heading elements are arranged in a hierarchy where the h1 element is followed by h2 elements which in turn are followed by h3 elements. This pattern would continue until one reaches the h6 elements. Think of heading elements as being similar to matryoshka dolls(the dolls that have layers such that opening up one doll leads to another smaller doll).

Shown below is an example of how the hierarchy of headings would work within a webpage:

<h1>Dogs</h1>

<h2>Breeds</h2>
<h3>Golden Retriever</h3>
<h3>Poodle</h3>

<h2>Dogs vs Cats</h2>
<h3>Dogs Are More Loving</h3>
<h3>Dogs Are More Loyal</h3>
The Paragraph Element

The paragraph element(denoted as <p>) contains text that make up a paragraph. The paragraph element, by default, creates space around it to seperate it from other nearby elements such as other pargraph elements.

Text Formatting Elements

An HTML webpage will almost always consist of text. Therefore, it is important to know ways that one can modify such text.

Listed below are some examples of text formatting elements:

  • Bold Text(<b></b>)
  • Italicized Text(<i></i>)
  • Subscript Text(<sub></sub>)
  • Superscript Text(<sup></sup>)
